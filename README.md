## English
A full-featured POS system for mini markets and shops with bilingual support, MMK/Baht currency format, product search, add to cart, receipt print, auto plus/minus summary, and colorful store UI/UX.

## မြန်မာ
ဒီ project သည် မီနီမားကတ်၊ ကုန်စုံဆိုင်နှင့် စတိုးဆိုင်များအတွက် ဖန်တီးထားသော POS system အပြည့်အစုံ ဖြစ်ပါသည်။

### ပါဝင်သော Features
- မြန်မာ / English language toggle
- MMK / Baht currency switch
- Product Search
- Camera Scan UI
- Product Cards
- Add to Cart
- Increase / Decrease Quantity
- Remove Items
- Subtotal / Tax / Total
- Auto အပေါင်းအနုတ် စာရင်းချုပ်
- Receipt / Invoice
- Print Receipt
- Daily Summary Dashboard
- Colorful Store Theme UI/UX
